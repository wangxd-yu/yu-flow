# 动态数据源管理

系统支持运行期（Runtime）通过底层 HikariCP 等连接池，实现**多数据库连接的动态注入与保活**。

## 场景概述
当您需要跨库联合查询，或者您的租户体系要求物理分离的独立 Database 时，不再需要在代码中编写繁琐的 `DataSourceBuilder`。

## 操作指南

在菜单的 【数据源管理】 界面中（路径 `/flow/dataSource`）：

1. **新建/编辑接入**：您可以录入**任意数据库驱动**（如 `mysql`, `highgo`）、URL（如 `jdbc:mysql://127.0.0.1...`）、用户名及密码。
2. **连接池心跳检测**：
   - 界面实时透出数据源**健康度**：绿灯代表正常连通，红灯（“异常”）代表远端数据库网络中断或口令失效。
   - 提供在线 **“测试连接”** 以沙盒运行拨测接口。
3. **全局控制**：通过状态开关（启用 / 禁用）硬性拦截相关 API 服务层对该 DB 的写操作，从而避免流量风暴。
