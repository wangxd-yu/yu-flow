# 页面可视化管理

在这一层，数据展现与表单操作最终面向客户——我们实现了强大的 **Amis 画布引擎**无缝拼接。

## 页面状态与生命周期

进入 【页面可视化管理】 (`/page-manage/list`)，您看到的是平台沉淀的每一张独立网页。例如“学生管理”、“数据源管理”列表。
每个页面都对应一个独特的 **访问路径**（例如 `/pages/student15`）。只要状态发布，终端用户即可通过该路由直达应用页面。

## 核心操作能力

- **设计 (Designer)**：  
  点击即可进入完整的 Amis 可视化设计器。您可以自由拖拽数据图表（表格、折线图、步骤条等）和输入组件。设计器直接在右侧配置面板挂接我们在前序章节生成的【动态 API】接口路径。
  > 备注：设计器的路由地址为独立放行路由 `/flow-ui/page-manage/designer/:id`。设计期间所有动作直接保存为 JSON schema 落盘。

- **预览 (Preview)**：  
  发布前的沙盒通道。点击可以即时呈现当前画布真实渲染效果。
  > 其路由位于 `/flow-ui/page-manage/preview/:id`。

- **克隆与回滚**：  
  基于深拷贝实现整个视图状态的快速复制，支撑版本迭代（“草稿”状态支持任意调试，发布即线上生效）。
